March 20, 2003
I'm sure most of you have read the news on the WiA site in regards to Khaile's current employment situation along with the game server going offline on March 21st. Don't worry though, it's only temporary (we hope) and I'm sure Khaile will find some kind of solution, but I believe it's more important that he finds another job because everyone in the working world knows you can't pay the bills sitting at home!

Aside from the bad news, the good news is that PA13 is showing good signs of development. Some of the new features (taken from the website) include:

Major Features
· New tileset system for world areas
· Active skills and spells
· 800x600 resolution
· Player classes
· Player-killing rules
· Much larger world

Minor Features and Bugfixes
· New user interface
· More complex quests
· Horses doesn't wander off so far
· Changed how monsters are added to the client (allows more monsters)

So what happens to the Akarrian Encyclopedia? More updates of course! While we wait for the game server to return to life and spring back into action, I will be adding more content so check back from time to time and......................go get a job Khaile :)
